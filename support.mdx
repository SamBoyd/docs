## FAQ

<AccordionGroup>
  <Accordion title="What's the basic workflow in OpenBacklog?">
    Create **Initiatives** (larger goals) → Break them into **Tasks** → Use AI assistance via MCP to refine and manage your work. The platform helps organize your development workflow with AI-powered insights.
  </Accordion>

  <Accordion title="How do I set up MCP integration?">
    Follow the [Claude Code setup guide](/mcp-integration/claude-code) to connect OpenBacklog with your IDE. You'll need to add the MCP server configuration to your Claude settings.
  </Accordion>

  <Accordion title="Is OpenBacklog free to use?">
    OpenBacklog is open source and free to self-host.
  </Accordion>

  <Accordion title="How do workspaces work?">
    Workspaces help organize different projects. Currently users are limited to a single workspace, though multi-workspace support is planned.
  </Accordion>

  <Accordion title="Where can I see what's planned?">
    Check the [GitHub issues](https://github.com/samboyd/openbacklog/issues) for upcoming features, or join the Discord to discuss the roadmap.
  </Accordion>
</AccordionGroup>
